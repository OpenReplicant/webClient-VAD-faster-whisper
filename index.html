Voice recognition server

Based on UrukHan's AI

  • Voice to text model: UrukHan/wav2vec2-russian
  • Text auto correction model: UrukHan/t5-russian-spell

    • Usage

    Send POST request to /recognize with audio data in .wav format

    Returns JSON:

    {
    "text": "текст распознанного аудио",
    "confidence": 0.9
}
