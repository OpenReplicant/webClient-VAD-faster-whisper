Voice recognition server

Based on UrukHan's AI

  • Voice to text model: UrukHan/wav2vec2-russian
  • Text auto correction model: UrukHan/t5-russian-spell

    • Usage

    Send POST request to /recognize with audio data in .wav format

    Returns JSON:

    {
    "transcripted_text": "russian text from transcripted audio", 
    "corrected_text": "corrected russian text"
}

    Example

    Based on this answer on StackOverflow 
    $ curl -XPOST -T speech.wav -H "Content-Type: audio/wav" http://localhost:3154/recognize
{
    "transcripted_text": "\u0440\u0430\u0441 \u0434\u0432\u0430 \u0442\u0440\u0438", 
    "corrected_text": "\u0420\u0430\u0437, \u0434\u0432\u0430, \u0442\u0440\u0438."
}
