Voice recognition server

Based on OpenAI whisper (faster-whisper)

Usage

Send POST request to /transcribe with audio data in .wav format

Returns JSON:

{
    "transcribed_segments": [
        {"text": "", "start": , "end": }
    ],
    "language": ""
}

Example

Based on this answer on StackOverflow 
$ curl -XPOST -T speech.wav -H "Content-Type: audio/wav" http://localhost:3157/recognize
{
    "transcribed_segments": [
        {"text": " 1, 2, 3", "start": 0.3, "end": 2.3}
    ], 
    "language": "ru"
}
